#1
Callback function er function sem kallar á annað function dæmi:
function a() {
    console.log('hello');
}
function b(function) {
    function();
}
b(a);

#2
Event loopið er loopið sem sér um að keyra öll events í réttri röð. 
Event loopið keyrir í gegnum allar skipanir sem þú gefur, og setur functions í call stackið þegar það er tómt.
